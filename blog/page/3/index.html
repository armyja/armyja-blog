    让 Ghost 支持 LaTeX 数学公式

    GhostLatex
    v1.0+ 版本的 Ghost 博客支持单篇文章的 Code Injection 功能，方便作者按需添加 JavaScript 或 CSS 代码，实现定制化。在 Code Injection 的 Post Footer 处粘贴 MathJax 脚本代码，即可支持 LaTeX 数学公式的显示。
